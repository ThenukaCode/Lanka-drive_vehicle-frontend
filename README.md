# Lanka Drive – Vehicle Selling Website Frontend

Lanka Drive is a modern and responsive vehicle selling website frontend developed to showcase vehicles with an attractive user interface and smooth user experience.

##  Features

- Responsive website design
- Modern vehicle showcase layout
- Navigation bar with smooth structure
- Vehicle cards and sections
- Attractive homepage design
- Mobile-friendly interface
- Interactive frontend components

##  Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap

##  Project Purpose

This project was created to improve frontend web development skills and demonstrate responsive UI/UX design concepts for a vehicle selling platform.
